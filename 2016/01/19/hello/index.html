如何部署hexo到github

准备工作:安装nodejs,git,因为在windows上这两样比较简单就略过了

安装hexo

因为hexo是node的一个包或模块,所以可以通过npm来安装.我们下面是全局安装,桌面空白处右键选择Git Bash Here

这样hexo就安装好了

报错

ERROR Plugin load failed: hexo-server

解决方法,执行命令:

浏览器输入http://localhost:4000即可,本地配置ok.

github新建仓库,点击New repository

Repository name和yourname.github.io,比如你的github是youda,那就是youda.github.io

进入你创建的目录里,找到_config.yml,编辑deploy字段,如下:
deploy:
type: git
repository: https://github.com/yourname/yourname.github.io.git
branch: master

上传到github

在浏览器输入https://yourname.github.io.git,yourname当然要改为你的用户名字.
